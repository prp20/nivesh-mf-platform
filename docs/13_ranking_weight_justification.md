# Ranking Weight Justification

## Philosophy
Reward consistency, downside protection, and manager skill over raw returns.

---

## Weight Breakdown
- Alpha: 25%
- Sortino Ratio: 20%
- Sharpe Ratio: 15%
- Rolling Return Consistency: 10%
- Upside Capture: 10%
- Downside Capture: -10%
- Std Deviation: -5%
- Beta: -5%
- TER: -5%
- AUM Stability: 5%

---

## Rationale
- Alpha reflects skill
- Sortino protects downside
- Capture ratios reveal market behavior
