# Analysis & Scoring Logic

## Metric Categories
1. Returns
2. Volatility & Risk
3. Risk-Adjusted Performance
4. Benchmark Regression
5. Market-Cycle Behavior
6. Cost & Scale

## Equity Fund Rules
- Sharpe & Sortino enabled
- Benchmark regression mandatory
- Capture ratios applied

## Ideal Fund Profile
- High Alpha, Sharpe, Sortino, R²
- Low Std Dev, Beta, TER
- Upside Capture > 1
- Downside Capture < 1

## Deterministic Scoring Model
Scores are computed using weighted metrics before any AI reasoning occurs.
