# Mutual Fund Management & Recommendation System

## Overview
This project is a finance-first, explainable mutual fund analytics and recommendation platform built using **FastAPI**, **Streamlit**, and **PostgreSQL/SQLite**.

The system focuses on:
- Correct financial analytics
- Transparent, explainable recommendations
- Controlled use of Agentic AI for reasoning (not computation)

## Tech Stack
- Backend: FastAPI
- Frontend: Streamlit
- Database: PostgreSQL (Prod), SQLite (Dev)
- ORM: SQLAlchemy
- AI Layer: Agentic LLM orchestration

## Core Capabilities
- Mutual fund & fund manager data management
- Daily & historical NAV ingestion
- Institutional-grade analytics
- Deterministic ranking engine
- AI-assisted explanation & comparison

## Guiding Principle
> Data → Metrics → Scoring → AI Reasoning → UI

This repository treats financial correctness as non-negotiable.
