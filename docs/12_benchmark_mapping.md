# Benchmark Mapping Rules

## Purpose
Define consistent benchmark assignment for each fund category.

---

## Equity Categories
- Large Cap → NIFTY 100 / NIFTY 50
- Mid Cap → NIFTY Midcap 150
- Small Cap → NIFTY Smallcap 250
- Flexi Cap → NIFTY 500

---

## Rules
- Each fund must have exactly one benchmark
- Benchmark changes are versioned
- Historical metrics use historical benchmark

---

## Usage
Benchmarks are mandatory for:
- Alpha
- Beta
- R-Squared
- Capture Ratios
