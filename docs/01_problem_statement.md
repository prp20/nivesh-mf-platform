# Problem Statement

Retail investors lack access to transparent, risk-aware tools to evaluate and compare mutual funds objectively.

### Key Issues
- Over-reliance on absolute returns
- Limited understanding of risk-adjusted metrics
- Opaque recommendation methodologies
- Overuse of AI without financial rigor

### Objective
Build a system that:
1. Stores and manages mutual fund data reliably
2. Computes institutional-grade performance & risk metrics
3. Produces explainable, deterministic fund rankings
4. Uses AI only for reasoning and narrative explanation
